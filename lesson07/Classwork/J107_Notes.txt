===== INTRODUCTING ARRAYS ===
Array is a data strucutre that represents a collection of the same types of data.

EX1.
double[] myList = new double[9];
- Array reference variable = myList
- Array element starts at index 0 (10 elements = 0-9)

===== DECLARING ARRAY VARIABLES ===
datatype[] arrayRefVar;

EX2.
double[] myList;

===== CREATING ARRAYS ===
datatype[] arrayRefVar = new datatype[arraySize];

EX3.
double[] myList = new double[9];

These fixed-sized arrays cannot change the array size since they only point to a reference in memory and not an actual value.

===== THE LENGTH OF THE ARRAY ===
Once an array is created, its size is fixed. It cannot be changed. You can find the size using arrayRefVar.length

EX4.
myList.length
10

===== DEFUALT VALUES ===
When an array is created, its elements are assigned the default value of:
- 0 for the numeric primitive data types (int, double)
- '\u0000' for char types (is an byte-sized integer)
- 'false' for boolean types ('true' is 1)

EX5.
- int[] arraylist1 = new int[9]
  arraylist[i] = 0
- double[] arraylist2 = new double[9]
  arraylist[i] = 0.0
- char[] arraylist3 = new char[9]
  arraylist[i] = 0 (\u0000)
- boolean[] arraylist4 = new boolean[9]
  arraylist[i] = false

Any non-primitive datatypes, like String, will be null until they are declared.

EX6.
String[] arraylist = new String[10];
arraylist[i] = null

===== INDEXED VARIBALES ===
The array elements are accessed through the index. The array indices are 0-based, i.e., it starts from 0 to arrayRefVar.length - 1. Each element in the array is represented using the following syntax, known as the indexed variable.

arrayRefVar[index]

EX7.
double[] myList = new double[9]; // indexed from 0 - 9
myList.length = 10

===== USING INDEXED VARIABLES ===
After an array is created, an indexed variable can be used in the same way as a regular variable.

EX8. The following code adds the value in myList[0], myList[1], and myList[2]:
int[] myList = new int[3]          // Index = [ 0][ 0][ 0]
myList[0] = 8;                     // Index = [ 8][ 0][ 0]
myList[1] = 10;                    // Index = [ 8][10][ 0]
myList[2] = myList[0] + myList[1]; // Index = [ 8][10][18] (8 + 10 = 18)

===== ARRAY INITIALIZERS ===
EX9. Declaring, creating, and initializing in one step:
double[] myList = new double[4]          // Index = [0.0][0.0][0.0][0.0]
double[] myList = {1.9, 2.9, 3.4, 3.5};  // Index = [1.9][2.9][3.4][3.5]

The shorthand syntax must be in one statement.

EX10. The shorthand is equivalent to this:
double[] myList = new double[4]          // Index = [0.0][0.0][0.0][0.0]
myList[0] = 1.9;                         // Index = [1.9][0.0][0.0][0.0]
myList[1] = 2.9;                         // Index = [1.9][2.9][0.0][0.0]
myList[2] = 3.4;                         // Index = [1.9][2.9][3.4][0.0]
myList[3] = 3.5;                         // Index = [1.9][2.9][3.4][3.5]

===== RANDOM SHUFFLING ===
for (int i = 0; i < myList.length - 1; i++) {
  // Generate an index j randomly
  int j = (int)(Math.random() * myList.length);
  
  // swap myList[i] with myList[j]
  double temp = myList[i];
  myList[i] = myList[j];
  myList[j] = temp;
}

===== ENHANCED FOR LOOP (FOR-EACH LOOP) ===
JDK 1.5 introduced a new for loop that enables you to traverse the complete array sequentially w/out using an index variable.

Ex. the following code displays all elements in the array myList
for (double value : muList)
  system.out.println(value);

EX. In general the syntax is:
for (elementType value: arrayRefVar)
  // Processing

You still have to use an index variable if you wish to traverse the array in a different order or chnage the elements in the array.
